Alternative ISA Methodology

# 8 Registers R0-R7 (8-bit)
# No flags, uses branches

# 4-byte fixed encoding length:
# Byte 0: Opcode + Mode (4 bits opcode + 2 bits addressing mode + 2 bits operand width)
#     Addressing Modes:
#     0    = no operand                 - NOP, RET, HALT
#     1    = immediate, symbol          - LD/ST Rx, Val     
#     2    = register-to-register       - LD/ST Rx, Ry      
#     3    = absolute mem addr          - LD/ST Rx, Addr    
#     4    = indirect through register  - LD/ST Rx, [Ry]    
#     5    = base pointer + offset      - LD/ST Rx, [Ry+Off]
#     Operand Widths:
#     0    = 8-bit (ignore byte 3)
#     1    = 16-bit
# Byte 1: Destination register (4 bits) + Source register (4 bits)
# Byte 2: Operand / low byte (immediate value, low addr byte, or offset)
# Byte 3: Operand / high byte (extended immediate, high addr byte, optional)
